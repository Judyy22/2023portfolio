React 로 구성된 Project

2022.03 - 2022.04

http://www.woodus.net

우드어스 협동조합 홈페이지 프로젝트에 참여하여 UI/UX 디자인과 프론트엔드 개발을 수행했습니다. 이 프로젝트는 백엔드 개발자와 함께 협업하여 실현한 결과물로, 사용자 경험을 개선하고 정보를 효과적으로 공유하는 역할을 수행하였습니다.
1.UI/UX 디자인
Figma와 같은 디자인 도구를 활용하여 웹사이트의 시각적 디자인 및 사용자 경험을 설계하였습니다.
2. 프론트엔드 개발 및 백엔드와 협력
React를 사용하여 디자인 시안을 웹 페이지로 구현하였습니다. 이를 통해 사용자들은 웹사이트를 손쉽게 탐색하고 상호작용할 수 있었습니다.백엔드 개발자와 협력하여 데이터 흐름 및 API 통신을 조절하였습니다. 이를 통해 데이터의 신뢰성과 일관성을 유지하고 사용자에게 실시간 정보를 제공하였습니다.

이 프로젝트는 협동조합과 관련된 정보를 효과적으로 공유하고 회원과 관심있는 이들에게 소통할 수 있도록 도와주는 결과물로서, UI/UX 디자인과 프론트엔드 개발이 효과적으로 결합하여 프로젝트의 성공에 기여하였습니다.
