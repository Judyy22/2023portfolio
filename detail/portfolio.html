HTML, CSS, JS 로 구성된 Project

2023.10.13 - 2022.10.14

https://judyy22.github.io/2023portfolio/

저는 웹 개발과 디자인에 대한 폭넓은 경험을 가지고 있으며, 이러한 경험들을 포트폴리오로 제작하여 정리한 웹페이지를 소개합니다. 이 포트폴리오는 여러 분야에서의 프로젝트 경험과 실무 경험을 다루고 있습니다.
1. HTML, CSS, Javascript 프로젝트 경험
저의 웹 개발 경력의 초기에는 HTML, CSS, JavaScript를 사용하여 간단한 웹 프로젝트를 개발했습니다. 이러한 경험을 통해 웹 개발의 기초를 다졌으며, 이러한 기초가 나중에 더 복잡한 프로젝트를 수행하는 데 중요한 역할을 했습니다.실직적인 회사에서의 경험 또한 다루고 있으며, 이 경험을 통해 협업 및 프로젝트 관리 능력을 발전시키고 웹 개발의 전문성을 강화했습니다.
2. React 사용 및 개인 프로젝트
개인적으로 React를 사용하여 프로젝트를 개발한 경험이 포함되어 있습니다. 이 프로젝트에서는 React를 활용하여 웹 애플리케이션을 개발하고 사용자 인터페이스를 구현했습니다. 또한, 이 웹페이지의 디자인 역시 개인적으로 수행한 부분으로, 사용자 경험을 최적화하기 위한 노력을 기술하고 있습니다.
포트폴리오를 통해 웹 개발 및 디자인에 대한 다양한 경험과 역량을 강조하며, 세부 사항을 더 자세히 표현할 수 있도록 노력한 결과물입니다. 계속해서 역량과 경험을 키우며 미래의 웹 프로젝트에 기여할 준비를 하고 있습니다.

이 포트폴리오 웹페이지는 자세한 내용을 확인할 때, 'Close' 버튼을 클릭했을 때 이전 상태로 돌아가는 기능을 제공합니다. 이는 세션(session)을 사용하여 구현되었으며, 사용자 경험을 향상시키고 편의성을 제공합니다.
